# Ethara Seat Allocation & Project Mapping System

## Overview

A full-stack application developed using React.js, FastAPI, and SQLite for managing employee seat allocation, project mapping, dashboard analytics, and an AI assistant.

---

## Features

- Employee Management
- Project Management
- Seat Allocation
- Dashboard Analytics
- AI Assistant
- Search Employees
- Search Seats
- Project Mapping

---

## Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios

### Backend
- FastAPI
- SQLAlchemy

### Database
- SQLite

---

## APIs

- Employee APIs
- Project APIs
- Seat APIs
- Dashboard APIs
- AI Assistant API

---

## Run Frontend

```bash
cd frontend
npm install
npm run dev
```

---

## Run Backend

```bash
cd backend
uvicorn main:app --reload
```
