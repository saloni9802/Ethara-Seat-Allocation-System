# AI_PROMPTS.md

## Prompt 1 - Architecture
Design a full-stack Ethara Seat Allocation & Project Mapping System using React, FastAPI, SQLAlchemy and SQLite.

## Prompt 2 - Database
Generate SQLAlchemy models for Employees, Projects, Seats and Seat Allocations with proper relationships.

## Prompt 3 - Backend APIs
Create REST APIs using FastAPI for employee management, project management, seat allocation, dashboard and AI assistant.

## Prompt 4 - Seat Allocation Logic
Implement seat allocation ensuring one employee has only one active seat and one seat can only be assigned to one employee.

## Prompt 5 - AI Assistant
Build a keyword-based AI assistant capable of answering employee seat location, project assignment and seat availability queries.

## Prompt 6 - Frontend
Create a responsive React + Tailwind dashboard with Employee, Project, Seat Allocation and AI Assistant pages.

## Prompt 7 - Testing
Test all CRUD APIs, dashboard endpoints and seat allocation flow using Swagger and frontend integration.

## Prompt 8 - Debugging
Fix API errors, React routing issues, database relationship problems and frontend/backend integration issues.

## Prompt 9 - Deployment
Deploy the FastAPI backend and React frontend using Render and Vercel.

## Prompt 10 - Refactoring
Refactor the codebase for readability, modularity and maintainability.
